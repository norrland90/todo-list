# TODO app

![screenshot](resources/screenshot.png)

### Description

Simple todo app.

### Features

- Add items with description and priority.
- Edit items.
- Mark items as done.
- Remove item / clear all items.
- Search function to search for items.
- Filter by priority.
- Uses local storage.

### Built With

- HTML.
- CSS.
- Vanilla Javascript.
